> **Creed:** Any path widely endorsed by the majority has already forfeited its
> excess returns.

Great Expectations is an AI **skill** (a system prompt you drop into any LLM)
that forces anti-consensus, non-linear strategic planning — then runs every
idea through a completeness gate so radical plans survive the real world
instead of leaking.

## Why it exists

AI plans grand things badly: it averages the future instead of breaking it.
This skill hard-codes the opposite — a deterministic pipeline with negative
constraints, not a chatbot that plans nicely.

## What it does

- **Four academic pillars** as hard constraints: breakthrough-innovation
  strategy, disruptive-innovation value systems, first-principles + Kahneman
  dual-system, and the BANI framework.
- **A forced, non-skippable workflow**: Deconstruct → Generate →
  **3.5 Completeness Mapping** → Vulnerability → Bias Audit → Falsifiable
  Assertions.
- **A completeness gate** that voids any path failing the stakeholder hologram,
  resource-antifragility, second/third-order, or spatiotemporal-redundancy test.
- **Eight-section output** you can audit, not vagueness you have to trust.

See the [full protocol →](docs/skill.html).

## Quick start

1. Copy [`SKILL.md`](SKILL.md) into a Custom GPT, a Claude Project, or prepend
   it as a system prompt.
2. Give it a grand goal, the industry's biggest "political correct" consensus,
   and your scarcest resource.
3. Or skip the copy-paste: open the **[Demo](docs/demo.html)**, fill three
   fields, and copy the composed prompt straight into your LLM.

## See it work

The demo includes real runs — including a [Google Glass back-test] that
pinpoints exactly which completeness check was missing when it failed.
