The skill · v3 completeness-forced

How it breaks things.

A deterministic pipeline with negative constraints — not a chatbot that plans nicely.

Foundations

Four academic pillars.

These are hard constraints, not decoration. Every plan must visibly draw on them.

P1

Breakthrough innovation

A strategic-management system covering Explore, Analyze, and Synthesize — not one step.

P2

Disruptive innovation

Value-system four dimensions: proposition, creation, delivery, realization. "Listen to customers" is the cage.

P3

First-principles + dual-system

Force System 2, suppress System 1. Decompose to irreducible elements; question every assumption on top.

P4

BANI framework

Brittle, Anxious, Non-linear, Incomprehensible. Plan in a non-linear environment, not Business-As-Usual.

Execution

Forced workflow — none may be skipped.

  1. Deconstruct. Refuse every "background constraint" unless it is a physical law. Break the goal into irreducible atomic units. Output an Assumption Inventory tagged CONSENSUS / SYSTEM-1 / ATOMIC.
  2. Generate. Produce at least two non-linear disruptive pathways. Each must state what it breaks and why a linear path fails. No incremental phrasing.
  3. 3.5 — Completeness Mapping. A hard gate. Every pathway must survive four penetration tests before it may be output: (①) stakeholder hologram, (②) resource-antifragility check, (③) second- & third-order effect chains, (④) spatiotemporal redundancy check. Failed paths are auto-voided.
  4. Vulnerability stress test. Simulate the plan's collapse under BANI shocks. Find the least-monitorable internal cause of a year-3 death. Provide the kill-switch.
  5. Cognitive bias audit. Scan for overconfidence, confirmation bias, anchoring. Label each with the bias name and where it hides.
  6. Falsifiable assertions. End with at least three claims reality could disprove, plus the mandated "if this fails, it was because of these 3 cognitive errors."

Negative constraints

Banned vocabulary.

Any occurrence forces a restart of the paragraph:

sustainable development · organic integration · steady progress · win-win for all · holistic synergy. Chinese equivalents (可持续发展 / 有机结合 / 稳步推进 / 多方共赢) are banned too.

Negative constraints

Forced specificity.

Say "leverage AI" and you must answer: which department's which manual work-hour is cut? If you cannot answer, the plan is invalid. No linear extrapolation as a starting point. No balancing all stakeholders by default — name who is sacrificed.

Output

Eight mandatory sections.

  1. Assumption Inventory
  2. First-Principles Decomposition
  3. Disruptive Pathways (≥2)
  4. Completeness Mapping (4 tests)
  5. Completeness Stress-Test Report
  6. Vulnerability Stress Test
  7. Cognitive Bias Audit
  8. Falsifiable Assertions (≥3)
